# Changelog

All notable changes to the **Sabr Tasbih** project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/) and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-13

### Added
*   **Core Tasbih Counter:** Animated progress indicators, mechanical clicks, haptics, undo, and reset confirmations.
*   **JSON Dua Library System:** Parsed from structured assets, categorized in 20+ types, supporting favorites, searches, and mini counters.
*   **Built-in Dhikr Seeding:** Seeds initial core prayers (SubhanAllah, Alhamdulillah, etc.) on first run.
*   **Dhikr Garden Gamification:** Virtual garden rendering palms, shrubs, jasmine, and fountains based on XP points.
*   **Streaks & Khatm Milestones:** Streak indicators and long-term project planners.
*   **Smart Reminders:** Custom alarms linked to counter entries.
*   **Offline-First & Cloud Sync:** Integration with local Hive databases and Firebase Cloud Firestore.
*   **PWA Integrations:** Custom manifests and service worker files.
*   **Testing Suites:** Unit and widget test frameworks.
*   **Full Documentation:** README, Contributing guidelines, Project Documentations.
